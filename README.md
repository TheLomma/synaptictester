# Synaptic Tester v1.0

Eine Web-App zum Testen der **Synaptic**-Fernbedienung von Conjuring Lab.

## Features

- **Calendar Prodigy**: Generiert ein zufälliges Datum innerhalb eines wählbaren Zeitraums und enthüllt schrittweise Jahr, Monat, Tag, Wochentag, vergangene Tage (und optional Sternzeichen)
- **Sprachausgabe**: Automatischer, zeitgesteuerter Ablauf mit konfigurierbaren Pausen
- **Hell- / Dunkelmodus**: Umschaltbar per Knopf, Auswahl wird gespeichert
- Mehrsprachig erweiterbar (weitere Tabs geplant)

## Deployment

### GitHub + Vercel (empfohlen)
1. Dieses Repository auf GitHub hochladen
2. Auf [vercel.com](https://vercel.com) einloggen → **New Project**
3. GitHub-Repository importieren
4. Framework: **Other** (statische HTML-Seite)
5. Deploy klicken — fertig!

### Lokal testen
Einfach `index.html` im Browser öffnen.
